# VIDEO-LIB


This is a fully functional video library app build using redux toolkit and firebase containing the below features

Hosted Link:

- Home Page
- Video Listing Page
- Playlist Management
- Like/Dislike
- Watch Later
- History
- Authentication
   - Sign up
   - Login
   - Logout
 - Single Video Page
 - Search Video
